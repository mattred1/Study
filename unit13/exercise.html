Array exercises

printReverse()

Write function printReverse() that takes an array as an argument and prints out the elements in the array in reverse order (Don't actually reverse the array itself).

isUniform()

Write a function isUniform() which takes an array as an argument and returns true if all the elements in the array are idential.

sumArray()

Write a function sumArray() that accepts an array of numbers and returns the sum of all the numbers in the array.

max()

Write a function max() that accepts an array of numbers and returns the maximum number in the array.
